Spinning Tower - Unity 3D

Spinning Tower is a 3D tower defence where the protagonist tries to grasp the meaning of life. You will control the rotation and fire of an enchanted cannon which rotates on the tower axis. Waves of kamikaze enemies constantly siege: enemies can both deal damage and apply malus to the tower. The game is planned to be released in 2025.

Cloudy Mess - Unity 3D

Cloudy Mess is 2D.5 puzzle game in early stage of development. The game is set in a tower full of puzzles: the main objective is to reach the top, in order to unravel all mysteries that the tower holds. More about this game will come out in next months.

Sadie - Unity 3D

Sadie is a 2D dungeon crawler game developed for the Global Game Jam 2024. The game features power ups and 3 different types of enemy and turn based interactions on a hexagonal grid.

Brain Bound - Unity 3D

Disdice - Unity 3D

Disdice is a simple 2D top down shooter, where a 6 faces die fights against other type of dice. The main character has 6 forms, each one with different stats and weapons; the player can change form, but he can't decide which form play. This game was our entry for the Bay Area Jam.
